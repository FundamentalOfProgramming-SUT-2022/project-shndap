- CLI based
- No graphics
- Commands:
  - ```createfile```: Create file
  - ```insertstr```: Insert
  - ```cat```: Cat
  - ```removestr```: Remove
  - ```copystr```: Copy  (built-in clipboard)
  - ```cutstr```: Cut   (built-in clipboard)
  - ```pastestr```: Paste (built-in clipboard)
  - ```find```: Find
  - ```replace```: Replace
  - ```grep```: Grep
  - ```undo```: Undo
  - ```auto-indent```: Closing pairs
  - ```compare```: Text comparator
  - ```tree```: Directory tree
  - ```=D```: Arman
